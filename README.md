# Power BI Insurance Premium Analysis Dashboard 📊

## Project Overview

This Power BI project analyzes insurance premium data to generate business insights and performance trends.

## Key Features

✔ Premium & maturity insights
✔ ROI (CAGR) calculations
✔ Region-based filtering using Row Level Security (RLS)
✔ Interactive dashboards
✔ Business intelligence visualizations

## Insights Covered

* Insurance premium distribution
* Investment maturity analysis
* ROI performance (CAGR)
* Region-wise analysis
* Customer segmentation insights

## Tools Used

* Power BI
* Data Modeling
* DAX Calculations
* Power Query

## Skills Demonstrated

* Data Visualization
* Business Intelligence
* Dashboard Design
* Data Analysis
* DAX Measures
